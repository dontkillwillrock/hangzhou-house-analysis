# 实验6：杭州学区房数据分析与房价预测

## 项目概述

本项目对杭州市学区房市场数据进行全链路数据分析与挖掘，覆盖**数据探索→特征工程→模型训练→调优评估→业务报告**完整流程。通过多种机器学习方法构建房价预测回归模型，帮助购房者评估学区房价格合理性，辅助投资决策。

- **数据规模：** 约 9,200 条二手房记录，26+ 特征字段
- **目标变量：** 单价（元/平米）— 回归任务
- **最佳模型：** 随机森林（R² ≈ 0.84）、Stacking 融合（R² ≈ 0.84）

## 数据说明

原始数据集包含约 9,200 条杭州市二手房交易记录（含学区房信息），主要字段如下：

| 字段 | 类型 | 说明 |
|------|------|------|
| 建筑面积 | 数值 | 房屋面积（平方米） |
| 户型 | 文本 | 如"3室2厅" |
| 楼层 | 类别 | 低层 / 中层 / 高层 |
| 总层数 | 数值 | 建筑总层数 |
| 朝向 | 类别 | 南 / 南北 / 东南 等 |
| 建筑年代 | 数值 | 建造年份 |
| 单价（元/平米） | 数值 | **目标变量** |
| 总价（万元） | 数值 | 房屋总价 |
| 所在学校 | 文本 | 对应学校名称 |
| 距离学校 | 文本 | 与学校距离描述 |
| 所在区县 | 类别 | 区域位置 |
| 学校属性 | 二值 | 区重点 / 市重点 / 名校附属 / 双语 等 |

### 特征工程

基于原始数据构造了以下衍生特征：

| 衍生特征 | 说明 |
|----------|------|
| 室 / 厅 | 从户型文本正则提取 |
| 距离学校\_米 | 距离文本转数值 |
| 房龄 | 2026 − 建筑年代 |
| 楼层\_数值 | 低层=1, 中层=2, 高层=3 |
| 单位面积房间数 | 室 / 建筑面积 |
| 学校等级评分 | 区重点+市重点+名校附属+双语 求和 |
| 学校特色数量 | 体育+艺术+语文+科技+外语 求和 |
| 楼层比例 | 楼层数值 / 总层数 |
| 每室平均面积 | 建筑面积 / 室 |
| 房龄等级 | 1=次新房 ~ 4=老旧 |
| 总价单价比 | 总价×10000 / (单价×面积) |

## 模型说明

### 各模型原理简介

**① 线性回归（基线模型）**
用一条直线拟合房价与特征的关系。假设建筑面积越大、房龄越新→房价越高。简单、可解释性强，但只能捕捉线性关系。R² ≈ **0.55**。

**② 随机森林（主力模型）**
建很多棵决策树（每棵树像一套"如果…就…"的规则），每棵树在不同样本子集上训练，最终预测取所有树的平均值。多棵树"投票"能防止过拟合，效果好且稳定。R² ≈ **0.84**。

**③ 梯度提升（主力模型）**
一棵一棵串行建树，后一棵去纠正前一棵的预测误差（残差）。步步为营，精度高，但训练慢于随机森林。R² ≈ **0.83**。

**④ Stacking 融合（最终模型）**
把随机森林、梯度提升等多个模型的预测结果当"新特征"，再训练一个 Ridge 回归作为"裁判"来决定如何组合它们。取各家之长，稳定可靠。R² ≈ **0.84**。

### 模型对比

| 模型 | 类型 | 原理 | 测试集 R² |
|------|------|------|-----------|
| 线性回归 | 线性模型 | 一条直线拟合 | ~0.55 |
| 随机森林 | Bagging 集成 | 多棵树投票取平均 | **~0.84** |
| 梯度提升 | Boosting 集成 | 逐棵补残差 | ~0.83 |
| Stacking 融合 | 模型融合 | 多个模型组合 + 元学习器 | ~0.84 |

### 评估指标

| 指标 | 说明 | 目标 |
|------|------|------|
| **R²** | 决定系数，越接近 1 越好 | > 0.80 |
| **RMSE** | 均方根误差，越小越好 | — |
| **MAE** | 平均绝对误差，越小越好 | — |
| **MAPE** | 平均绝对百分比误差 | < 15% |

### 关键技术

- **数据清洗：** 缺失值中位数/众数填补、异常值截断
- **编码策略：** 独热编码（朝向、所在区县）、有序编码（楼层）
- **特征缩放：** StandardScaler（训练集 fit → 测试集 transform）
- **交叉验证：** 5 折 R² 评估
- **超参调优：** GridSearchCV 搜索最佳参数组合
- **模型融合：** Stacking（基学习器：随机森林 + 梯度提升 + 线性回归，元学习器：岭回归）
